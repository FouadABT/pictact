# PicTact Product Overview

PicTact is a real-time competitive image hunting game built as a Reddit Devvit app. Players compete to find and submit photos matching moderator-created challenges, with the first valid submission winning points and advancing on leaderboards.

## Core Game Mechanics
- **Challenge System**: Moderators create time-limited photo hunting challenges
- **Competition Format**: First player to submit a correct photo wins points
- **Validation Method**: Dual system with community voting + moderator override
- **Reward System**: Points, leaderboards, and collectible trophy badges
- **Multiplayer Scope**: Subreddit-based competitions with global rankings

## Target Audience
- Reddit communities looking for interactive engagement
- Photography enthusiasts and casual mobile users
- Subreddit moderators seeking community-building tools

## Key Value Propositions
- Transforms passive Reddit browsing into active competition
- Encourages real-world exploration and creativity
- Builds community engagement through shared challenges
- Provides moderators with powerful community interaction tools
